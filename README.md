## 🧑‍💻 Getting Started Locally

1. **Clone the repo**
```bash
git clone https://github.com/suaib-saifi/spotify-clone.git
cd spotify-clone
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the project**
```bash
npm run dev
```

4. Open in browser:  
`http://localhost:5173`
